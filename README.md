# ドコモ料金見積もり

ドコモショップ店頭（COヘルパー業務）用の料金シミュレーター＋見積書作成PWA。

**公開URL: https://hisapiyp622-alt.github.io/docomo-quote/**（スマホで開いて「ホーム画面に追加」でアプリとして使えます）

## 構成
- `index.html` / `style.css` / `app.js` — アプリ本体（素のHTML/CSS/JS）
- `data.js` — 料金マスタ初期値（**2026-07-10にドコモ公式を調査・相互検証した税込確定値**）
- `manifest.webmanifest` / `sw.js` / `icon.svg` — PWA（ホーム画面追加・オフライン動作）

## 機能
- **3パターン同時見積もり**（A/B/C切替・コピー・見積書にパターン比較表）
- **見積もりタブ**: 手続き種別 → プラン（現行: MAX/ポイ活MAX/ポイ活20/mini/ahamo/U15、旧: eximo/eximoポイ活/irumo/5Gギガホプレミア/ギガライト）→ 割引（みんなドコモ割・光/home 5Gセット割・dカードお支払割2段階・でんきセット割・長期利用割）→ 通話・**オプション・サービス（キャリア/代理店の区別なしの統合リスト・項目ごとに月額/契約時一括・補償は機種別金額を選択式）**→ 端末（一括/分割12〜36回/いつでもカエドキ＝残価・プログラム利用料対応）＋**アクセサリ（docomo select等・一括/分割12/24/36回）**→ 初期費用（手数料定番項目のチェック式）
- **期間セグメント計算**: 端末分割・アクセサリ分割・期間限定割引の切れ目ごとに月額を分割表示（見積書に「月額の推移」表）
- **見積書タブ**: お客様向けお見積書（印刷/PDF保存対応・ブラウザメニュー印刷でも最新内容を出力）。印刷は2ページ構成（1ページ目=プラン・オプション、2ページ目=端末代金・初期費用）
- **マスタ設定タブ**: 全単価の編集に加え、キャリアオプション/代理店サービス/手数料項目の**追加・削除・並び替え**（localStorage保存・初期値に戻す可）

## 料金データの前提（2026-07-10時点）
- 事務手数料: 店頭一律4,950円（2025-09-05改定。Web手続きは無料）
- みんなドコモ割3回線: 新プラン−1,210円 / 旧プラン−1,100円
- dカードお支払割: 新プラン −220円（GOLD系−550円）/ 旧プラン一律−187円
- カエドキ: 2026-03-05以降加入分は返却時プログラム利用料（機種により0〜22,000円、ドコモで買替えなら免除）
- 料金改定があったら「マスタ設定」タブで修正するか、`data.js` を更新
